# Laboratorio_1_Python_Pandas_Matplotlib
Trabajo practico para el Primer Laboratorio de Especialización en Análisis de Datos 2025 - Informatorio Chaco

# Descripción del Proyecto
Este repositorio contiene un análisis exploratorio de datos (EDA) de los datasets de precios históricos del dólar, salarios históricos medianos y costos de la Canasta Básica Total de Argentina. El objetivo principal es identificar la evolución de los precios y salarios medidos en dólares y el poder adquistivo determinado por la cantidad de canastas que podía comprar un trabajador con un salario mediano en diferentes épocas, utilizando herramientas como Python y Pandas y Matplotlib. El proyecto demuestra el proceso de Extracción, Transformación y Carga (ETL), así como técnicas de visualización y agregación de datos para responder a preguntas de negocio.
